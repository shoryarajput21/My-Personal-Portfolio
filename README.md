### Run Locally

To run **Portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/portfolio.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/portfolio.git
```
